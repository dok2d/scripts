# Конвертер чатов Telegram (convert-tg-chats)

Директория содержит утилиту для преобразования экспортированных JSON-файлов Telegram в читаемый текстовый формат, удобный для последующего анализа или подачи в LLM для создания саммари.

## Содержимое

### `convert-tg-chats.py`

**Назначение:** Конвертация экспортированного JSON-файла чата Telegram в структурированный текстовый формат с группировкой по дням и информацией об участниках.

**Описание:**
Скрипт обрабатывает JSON, полученный через экспорт данных Telegram (например, `result.json`), и создаёт текстовый файл, где каждое сообщение представлено в виде:

```
[ЧЧ:ММ] U01: Текст сообщения [🔥×2] ↩U02:«Цитируемый текст»
```

**Основные возможности:**
- Нумерация активных участников (U01, U02, ... по убыванию количества сообщений)
- Разделители по дням (`── 31.12.2023 ──`)
- Фильтрация по дате
- Включение/исключение реакций и ответов (реплаев)
- Дедупликация сообщений по ID
- Игнорирование сообщений без текста

**Использование:**
```bash
./convert-tg-chats.py result.json -o chat_summary.txt
```

**Параметры:**
- `input` — путь к JSON-файлу (обязательный)
- `-o, --output` — выходной файл (если не указан, вывод в stdout)
- `--from YYYY-MM-DD` — начальная дата
- `--to YYYY-MM-DD` — конечная дата
- `--no-reactions` — не включать реакции
- `--no-replies` — не включать цитаты ответов

**Примеры:**
```bash
# Весь чат с реакциями и ответами
./convert-tg-chats.py chat.json -o chat.txt

# Только за январь 2024, без реакций
./convert-tg-chats.py chat.json -o january.txt --from 2024-01-01 --to 2024-01-31 --no-reactions

# Вывод в консоль, только сообщения с текстом
./convert-tg-chats.py chat.json --no-reactions --no-replies
```

**Формат вывода:**
```
# Название чата

## Участники
U01 — Иван Иванов
U02 — Петр Петров
U03 — Сидор Сидоров

## Сообщения

── 01.01.2024 ──
[10:15] U01: Всем привет! [👍×3]
[10:16] U02: Привет! ↩U01:«Всем привет!»
[10:20] U03: И вам не болеть

── 02.01.2024 ──
[09:00] U01: Новый день

# Итого: 4 сообщения
```

**Зависимости:**
- Python 3.6+ (только стандартная библиотека, дополнительные пакеты не требуются)
