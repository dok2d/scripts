# Личная коллекция скриптов

Данный репозиторий содержит коллекцию скриптов и вспомогательных файлов для автоматизации различных задач: от миграции репозиториев и конвертации видео до работы с сетями и создания плейлистов.

## Структура

- **Корневая директория** — основные утилиты и скрипты общего назначения
- **`depricated/`** — устаревшие скрипты, сохранённые для истории
- **`llm/`** — утилиты для работы с большими языковыми моделями
- **`misc/`** — разное (развлекательные и вспомогательные скрипты)
- **`network/`** — сетевые утилиты (работа с IP-адресами и подсетями)
- **`nginx_autoindex/`** — генерация плейлистов для nginx и видео/аудио плееров
- **`support_files/`** — вспомогательные файлы и списки
- **`web/`** — веб-ориентированные скрипты (загрузка контента, мониторинг)
- **`convert-tg-chats/`** — конвертер экспортированных чатов Telegram

---

Подробное описание каждого скрипта находится в соответствующих директориях.
