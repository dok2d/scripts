# Личная коллекция скриптов

Данный репозиторий содержит коллекцию скриптов и вспомогательных файлов для автоматизации различных задач: от миграции репозиториев и конвертации видео до работы с сетями и создания плейлистов.

## Структура

- **Корневая директория** — основные утилиты и скрипты общего назначения
- **`depricated/`** — устаревшие скрипты, сохранённые для истории
- **`llm/`** — утилиты для работы с большими языковыми моделями
- **`misc/`** — разное (развлекательные и вспомогательные скрипты)
- **`network/`** — сетевые утилиты (работа с IP-адресами и подсетями)
- **`nginx_autoindex/`** — генерация плейлистов для nginx и видео/аудио плееров
- **`support_files/`** — вспомогательные файлы и списки
- **`web/`** — веб-ориентированные скрипты (загрузка контента, мониторинг)
- **`convert-tg-chats/`** — конвертер экспортированных чатов Telegram

## Скрипты в корневой директории

### `git_org_mirroring.sh`

**Назначение:** Массовое создание зеркал репозиториев между GitHub, GitLab и Gitea.

**Описание:**
Скрипт автоматизирует перенос всех репозиториев из организации/пользователя на одном хостинге (GitHub, GitLab, Gitea) в целевой Gitea с созданием зеркал. Поддерживает автоматическое определение типа источника по домену.

**Возможности:**
- Определение типа хостинга (GitHub, GitLab, Gitea) по API-энпоинтам
- Пагинация для организаций с большим количеством репозиториев
- Пропуск уже существующих репозиториев в целевой Gitea
- Настройка приватности создаваемых зеркал

**Зависимости:** `jq`, `curl`

**Использование:**
```bash
./git_org_mirroring.sh -e https://github.com/kubernetes -l https://gitea.example.com/myorg --private -t ghp_xyz123
```

**Параметры:**
- `-e, --source_link URL` — ссылка на исходную организацию/пользователя
- `-l, --target_link URL` — ссылка на целевую организацию в Gitea
- `-p, --private` — создавать приватные репозитории (по умолчанию публичные)
- `-t, --target-token TOKEN` — токен аутентификации для целевой Gitea
- `-h, --help` — показать справку

### `ffconv`

**Назначение:** Генерация и выполнение команд ffmpeg для массовой конвертации видео в MP4 (H.264).

**Описание:**
Скрипт находит все видеофайлы в указанной директории (или обрабатывает отдельный файл), анализирует их и генерирует оптимальные команды ffmpeg для конвертации в MP4 с сохранением структуры поддиректорий.

**Возможности:**
- Поддержка множества форматов: avi, mkv, ts, mov, flv, webm, wmv, m2ts, vob и др.
- Автоматический поиск внешних аудиодорожек и субтитров
- Поддержка GPU-ускорения (NVENC) через переменную окружения `FFCONV_GPU=yes`
- Выбор аудиодорожки по языку (русский, английский, японский) с возможностью сохранить несколько
- Очистка имени выходного файла от спецсимволов

**Переменные окружения:**
- `FFCONV_GPU=yes` — использовать аппаратное кодирование (h264_nvenc)
- `FFCONV_SOURCE_SUBS=yes` — сохранять субтитры из исходного файла
- `FFCONV_SOURCE_AUDIO=yes` — сохранять дополнительные аудиодорожки (английская или японская)

**Использование:**
```bash
# Конвертировать все видео в текущей директории
./ffconv

# Конвертировать из одной директории в другую
./ffconv исходная_папка целевая_папка

# Конвертировать один файл в указанную папку
./ffconv video.avi целевая_папка

# С аппаратным кодированием
FFCONV_GPU=yes ./ffconv
```

**Процесс работы:**
1. Поиск видеофайлов
2. Анализ каждого файла (кодек, цветовой профиль, аудиодорожки)
3. Генерация команд ffmpeg
4. Запрос подтверждения перед запуском
5. Последовательное выполнение конвертации
