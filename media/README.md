# Медиа-скрипты (media)

Директория содержит скрипты для обработки видео и аудиофайлов.

## Содержимое

### `ffconv`

**Назначение:** Генерация и выполнение команд ffmpeg для массовой конвертации видео в MP4 (H.264).

**Описание:**
Скрипт находит все видеофайлы в указанной директории (или обрабатывает отдельный файл), анализирует их и генерирует оптимальные команды ffmpeg для конвертации в MP4 с сохранением структуры поддиректорий.

**Возможности:**
- Поддержка множества форматов: avi, mkv, ts, mov, flv, webm, wmv, m2ts, vob и др.
- Автоматический поиск внешних аудиодорожек и субтитров
- Поддержка GPU-ускорения (NVENC) через переменную окружения `FFCONV_GPU=yes`
- Выбор аудиодорожки по языку (русский, английский, японский) с возможностью сохранить несколько
- Очистка имени выходного файла от спецсимволов

**Переменные окружения:**
- `FFCONV_GPU=yes` — использовать аппаратное кодирование (h264_nvenc)
- `FFCONV_SOURCE_SUBS=yes` — сохранять субтитры из исходного файла
- `FFCONV_SOURCE_AUDIO=yes` — сохранять дополнительные аудиодорожки (английская или японская)

**Использование:**
```bash
# Конвертировать все видео в текущей директории
./ffconv

# Конвертировать из одной директории в другую
./ffconv исходная_папка целевая_папка

# Конвертировать один файл в указанную папку
./ffconv video.avi целевая_папка

# С аппаратным кодированием
FFCONV_GPU=yes ./ffconv
```

**Процесс работы:**
1. Поиск видеофайлов
2. Анализ каждого файла (кодек, цветовой профиль, аудиодорожки)
3. Генерация команд ffmpeg
4. Запрос подтверждения перед запуском
5. Последовательное выполнение конвертации
