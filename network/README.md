# Сетевые утилиты (network)

Директория содержит Python-скрипты для работы с IP-адресами и подсетями.

## Содержимое

### `net_info.py`

**Назначение:** Расчёт информации о подсети (минимальный/максимальный адрес, количество доступных адресов).

**Описание:**
Скрипт принимает подсеть в формате CIDR и выводит:
- Минимальный адрес (первый доступный для хоста)
- Максимальный адрес (последний доступный для хоста)
- Количество доступных адресов (исключая адрес сети и широковещательный)

**Использование:**
```bash
./net_info.py 10.28.52.0/27
```

**С флагом `--value` можно получить только одно значение:**
```bash
./net_info.py 10.28.52.0/27 --value min     # 10.28.52.1
./net_info.py 10.28.52.0/27 --value max     # 10.28.52.30
./net_info.py 10.28.52.0/27 --value count   # 30
```

---

### `gen_netw.py`, `gen_netw2.py`, `gen_netw3.py`

**Назначение:** Агрегация IP-адресов в минимальные подсети с разными стратегиями.

**Общее описание:**
Все три скрипта читают из файла список IP-адресов/сетей (по одному на строку) и выводят агрегированные подсети. Различаются алгоритмами финального объединения.

**Входной формат:**
```
192.168.1.0/24
10.0.0.1/32
10.0.0.2/32
172.16.1.0/28
```

---

#### `gen_netw.py` (базовая агрегация)

**Описание:**
- Агрегирует IP-адреса в минимальные подсети
- Расширяет подсети с маской /29 или больше до /28
- Убирает дубликаты

```bash
./gen_netw.py ip_list.txt
```

---

#### `gen_netw2.py` (с финальным объединением)

**Описание:**
Всё то же, что и `gen_netw.py`, плюс дополнительный проход `collapse_addresses` для финального укрупнения подсетей.

```bash
./gen_netw2.py ip_list.txt
```

---

#### `gen_netw3.py` (агрессивное объединение)

**Описание:**
Самая "жадная" версия: после стандартной агрегации пытается объединить последовательные подсети в ещё более крупные, даже если они не являются смежными в строгом смысле (используется проверка `supernet_of`).

```bash
./gen_netw3.py ip_list.txt
```

**Пример работы:**

Входной файл:
```
192.168.1.1/32
192.168.1.2/32
192.168.1.3/32
192.168.2.1/32
192.168.2.2/32
```

`gen_netw.py`:
```
192.168.1.0/29
192.168.2.0/29
```

`gen_netw3.py` (агрессивный режим) может объединить их в `192.168.0.0/22`, если это возможно.
