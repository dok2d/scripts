# Веб-ориентированные скрипты (web)

Директория содержит скрипты для загрузки контента из интернета, мониторинга и автоматизации веб-задач.

## Содержимое

### `geter`

**Назначение:** Массовая загрузка файлов по списку URL с проверкой размера и пропуском уже скачанного.

**Описание:**
Скрипт читает файл `links` в текущей директории (по одному URL на строку) и скачивает каждый файл. Перед скачиванием проверяется:
- Существует ли уже файл с таким именем
- Если существует, сравнивается размер с удалённым (по заголовку Content-Length)
- Если удалённый файл больше, происходит перезагрузка (сначала во временный файл, потом замена)

**Особенности:**
- Файлы с расширением `.exe` или `.bin` сохраняются в поддиректорию `win/`
- Файлы с расширением `.sh` сохраняются в поддиректорию `lin/`
- Поддерживает декодирование URL-encoded имён (функция `decodeURL`)

**Использование:**
```bash
# Создать файл links со списком URL
echo "https://example.com/file.zip" > links
./geter
```

### `duolingowatch`

**Назначение:** Мониторинг активности пользователя Duolingo и отправка уведомлений в Telegram.

**Описание:**
Скрипт в бесконечном цикле опрашивает API Duolingo и отслеживает изменения опыта (XP) по курсам. При увеличении XP отправляет сообщение в Telegram.

**Настройка:**
В начале скрипта нужно задать:
- `tg_bot_key` — токен Telegram-бота
- `tg_chat_id` — ID чата для отправки

**Использование:**
```bash
./duolingowatch username
```

**Пример уведомления:**
```
🔮 username втихаря получил 15 за en(ru)
```

**Примечание:** Скрипт использует публичное API Duolingo, которое может измениться. Между запросами делается случайная пауза (20–90 секунд).

### `get_manga`

**Назначение:** Скачивание глав манги с сайтов, использующих определённую структуру.

**Описание:**
Скрипт рекурсивно проходит по главам манги, начиная с указанного URL, и скачивает все изображения страниц.

**Алгоритм:**
1. Определяет домен и путь к первой главе
2. Для каждой главы:
   - Создаёт директорию
   - Парсит страницу для получения списка изображений (ищет `readerInit`)
   - Скачивает все изображения, пропуская уже существующие
3. Переходит к следующей главе по ссылке `nextChapterLink`

**Использование:**
```bash
./get_manga "https://site.com/manga/title/chapter/1"
```

**Структура сохранения:**
```
./manga/title/chapter/1/001.jpg
./manga/title/chapter/1/002.jpg
...
./manga/title/chapter/2/001.jpg
```

Также создаётся файл `link.txt` с исходным URL для справки.

### `get_autoinspect`

**Назначение:** Обёртка для `geter`, автоматически собирающая ссылки на файлы из HTML-листинга директории.

**Описание:**
Скрипт принимает URL, парсит HTML-страницу, извлекает все ссылки (кроме родительской директории) и передаёт их в `geter` для скачивания.

**Использование:**
```bash
./get_autoinspect "https://example.com/files/"
```

**Примечание:** Скрипт создаёт временный файл `links` и вызывает `geter` из той же директории.

### `get_gog_library`

**Назначение:** Заглушка/заметка.

**Описание:**
Скрипт не содержит кода, только комментарий с ссылкой на актуальный инструмент для скачивания игр с GOG.

**Содержимое:**
```bash
Move to: https://github.com/dok2d/gog-games-downloader
```
