# Вспомогательные файлы (support_files)

Директория содержит текстовые файлы со списками и конфигурациями, используемыми другими скриптами.

## Содержимое

### `git_repos`

**Назначение:** Список организаций и репозиториев для массового клонирования/зеркалирования.

**Описание:**
Текстовый файл, в котором каждая строка содержит имя организации или репозитория в формате `owner/repo`. Используется скриптами типа `git_org_mirroring.sh` для определения источников.

**Формат:**
- Простые имена организаций (например, `python`, `nginx`)
- Репозитории с указанием владельца (например, `pikvm/pikvm`)
- Возможны комментарии? (судя по наличию `#` в некоторых строках, да)

**Пример содержимого:**
```
0x90
ansible
pikvm/pikvm
w23
```
