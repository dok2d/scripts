# Разные скрипты (misc)

Директория содержит скрипты, не относящиеся к конкретной категории, включая развлекательные и вспомогательные.

## Содержимое

### `dvd_screen`

**Назначение:** Анимация логотипа, отскакивающего от границ терминала (как в старых DVD-плеерах).

**Описание:**
Скрипт создаёт в терминале анимацию логотипа, который движется по экрану и меняет цвет при ударе о границу. Размер терминала определяется автоматически. По умолчанию используется текст "DVD", но можно указать свой.

**Использование:**
```bash
./dvd_screen          # анимация с логотипом "DVD"
./dvd_screen "MUSIC"  # анимация с текстом "MUSIC"
```

**Управление:**
- Для выхода — `Ctrl+C`
- Цвет меняется при каждом отскоке (используются 8 стандартных цветов терминала)

### `ya-cover-gen`

**Назначение:** Генерация простой обложки (изображения) с текстом для Яндекс.Музыки или других целей.

**Описание:**
Скрипт использует ImageMagick для создания квадратного изображения с чёрным фоном, жёлтым текстом и красной рамкой. Шрифт — Montserrat-Black (должен быть установлен в системе).

**Зависимости:** ImageMagick (`convert`), шрифт Montserrat-Black

**Использование:**
```bash
./ya-cover-gen "Мой альбом"
```

**Результат:** Файл `output.png` в текущей директории.

**Настройки по умолчанию (можно изменить в скрипте):**
- Размер: 1000x1000
- Цвет фона: чёрный
- Цвет текста: `#ffd800` (жёлтый)
- Цвет рамки: красный
- Размер шрифта: 250
- Рамка: 30 пикселей
