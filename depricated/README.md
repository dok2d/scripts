# Устаревшие скрипты (depricated)

Данная директория содержит скрипты, которые больше не поддерживаются, но сохранены для исторической справки или возможного рефакторинга.

## Содержимое

### `giter`

**Назначение:** Массовое клонирование репозиториев GitHub организации и упаковка в RAR-архивы.

**Описание:**
Скрипт скачивает все публичные репозитории указанной организации (или один конкретный репозиторий) в формате `--bare` и создаёт многотомный RAR-архив. Архивирование происходит с максимальным сжатием и разбивкой на тома по 10 МБ.

**Зависимости:** `git`, `rar`, `jq`

**Переменные окружения:**
- `tmpbasedir` — временная директория (по умолчанию `/tmp`)
- `outpath` — директория для сохранения архивов (по умолчанию `/var/www/dwnld`)

**Пример использования:**
```bash
# Архивировать все репозитории пользователя kubernetes
./giter kubernetes

# Архивировать конкретный репозиторий
./giter kubernetes/kubernetes
```

**Примечание:** Скрипт автоматически устанавливает недостающие пакеты через `apt-get` и меняет владельца на `www-data`, если целевая директория находится в `/var/www/`.

### `get_gog_library`

**Назначение:** Заглушка/заметка.

**Описание:**
Скрипт не содержит кода, только комментарий с ссылкой на актуальный инструмент для скачивания игр с GOG.

**Содержимое:**
```bash
Move to: https://github.com/dok2d/gog-games-downloader
```
